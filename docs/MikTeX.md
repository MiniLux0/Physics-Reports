# 🧱 PARTE 1 — Instalación de LaTeX (MiKTeX)

Esta sección se enfoca únicamente en preparar el sistema para poder compilar archivos `.tex`.

👉 Al terminar esta parte podrás:

* Abrir archivos `.tex`
* Compilarlos a PDF
* Instalar paquetes automáticamente cuando falten

---

# 1️⃣ Instalar MiKTeX (motor LaTeX)

MiKTeX es el programa que convierte código LaTeX en PDF.

### 📥 Descarga

Ve a:
[https://miktex.org/download](https://miktex.org/download)

Descarga la versión para Windows.

---

### ⚙️ Instalación

Durante la instalación:

* Usa opciones por defecto
* “Next → Next → Install”
* No cambies rutas si no sabes lo que haces

---

### 🔄 Actualización obligatoria

Después de instalar:

1. Abre **MiKTeX Console**
2. Ve a la pestaña **Updates**
3. Ejecuta:

   * **Check for updates**
   * **Update now**

⚠️ Esto evita errores como paquetes faltantes o compilación fallida.
